# PromptProxy

**Web LLM のためのローカル AI コーディングツール・プロキシ**

PromptProxy は、強力なローカル AI コーディングツールである **Aider** と、**Web LLM**（Gemini 1.5 Pro や Claude 3.5 Sonnet など）の巨大なコンテキストウィンドウを橋渡しするための GUI アプリケーションです。

ブラウザ上の AI インターフェースを「LLM バックエンド」として活用しながら、Aider のリポジトリ解析能力とパッチ適用機能を利用することができます。

## 主な特徴

- **モジュラー・ワークフロー**: Repo Map 生成、XML ファイルパッキング、パッチ適用のための独立したブロック構成。
- **アップロード制限の回避**: 巨大なソースファイルを自動で行単位のチャンクに分割し、Web UI へ簡単にドラッグ＆ドロップ可能。
- **オフライン・パッチ適用**: AI が生成した SEARCH/REPLACE ブロックを、外部 API 接続なしでローカルファイルに適用。
- **改行コードの保護**: ファイル本来の改行コード（LF/CRLF）を自動で判別し、維持したまま保存。
- **ポータブル**: 設定はすべてローカルの `config.json` に保存されます。

## 必要条件

- **Node.js** (v18 以降)
- **Rust** (Stable)
- **pnpm** (推奨)
- **Aider** (インストール済みで、CLI からアクセス可能であること)

## 使い方

### 1. インストール

リポジトリをクローンし、依存関係をインストールします：

```bash
pnpm install
```

### 2. 開発モード

開発モードでアプリケーションを起動します：

```bash
pnpm tauri dev
```

### 3. ビルド

製品版の実行ファイルを作成します：

```bash
pnpm tauri build
```

実行ファイルは `src-tauri/target/release` ディレクトリ内に生成されます。
